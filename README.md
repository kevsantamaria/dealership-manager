# dealership-manager

Dealership Manager is a web-based application designed to manage an automobile dealership's inventory, allowing administrators to handle vehicles for sale, their specifications, and availability. The system follows a frontendâ€“backend architecture, built with React and Tailwind CSS on the frontend, Express on the backend, and a PostgreSQL database, and it also uses direct SQL queries (no ORM) for database access.

## Run backend

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

## Run frontend

To install dependencies:

```bash
bun install
```

To start a development server:

```bash
bun dev
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
